# 2017 Honda Civic LX Grey Showcase

A web page dedicated to the 2017 Honda Civic LX in grey, showcasing its specifications, features, photos, and owner insights.

## File Structure
- `index.html`: Main HTML file with page structure.
- `css/style.css`: Styles for layout, responsiveness, and aesthetics.
- `js/main.js`: JavaScript for lightbox gallery and form handling.
- `images/`: Contains images (civic-front.jpg, civic-side.jpg, civic-interior.jpg, civic-rear.jpg).

## Setup
1. Save all files in the specified structure.
2. Add images to the `images/` folder (ensure they are named as listed or update paths in `index.html`).
3. Open `index.html` in a browser to view the page.
4. Host on a server (e.g., GitHub Pages, Netlify) for online access.

## Notes
- Images are placeholders; replace with actual photos of a 2017 Honda Civic LX in grey.
- The contact form uses client-side validation and alerts; for production, integrate a backend (e.g., Node.js, Firebase) for form submissions.
- Tested on modern browsers (Chrome, Firefox, Safari).
